# Project Overview

This project includes two model implementations:

* **final-income-model.ipynb**
  This is the primary model used for our results and analysis. All reported outcomes in the project are based on this model.

* **two-stage-model.ipynb**
  This notebook is included for reference purposes only. It demonstrates the structure and implementation of a two-stage modeling approach but is not used in the final results.

## Notes

* A supporting scientific paper is included to explain the methodology and reasoning behind these models.
* For evaluation and conclusions, refer to the **final-income-model**.
